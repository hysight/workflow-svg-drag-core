# 开发

## 安装依赖

```
    npm install 
```

## 功能描述

```
    机器学习-点/线等拖拉拽工作流核心svg渲染原理部分demo
```

## 启动命令

```
    // 运行一次即可
    npm run dll

    // 每次启动需要运行
    npm run start
```
